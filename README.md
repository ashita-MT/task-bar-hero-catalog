# TBH: Task Bar Hero - Steam Market 装备图鉴

Steam 市场装备数据可视化图鉴，包含全部 10 种稀有度等级的装备信息。

## 功能
- 按稀有度分类查看装备（常见→宇宙）
- 搜索过滤
- 稀有度多选过滤
- 分页加载（查看更多）
- 属性中文翻译
